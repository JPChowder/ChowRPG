# RPG de Cartinhas do Chowder

Este é o repositório que o Chowder demorou toda uma graduação inteira pra criar e que facilita para que as pessoas possam consultar as regras desse sistema de RPG em uma estrutura de wiki.

## O Sistema

Este jogo faz uso de cartas de baralho ao invés de dados. Os jogadores e o mestre possuem um naipe inteiro menos o ás (cartas de 2 ~ 13) como sua fonte de realizar jogadas. As cartas são embaralhadas e deixadas com as faces para baixo, quando se vai tentar uma jogada basta virar a carta que está sobre o monte. Note que isso remove a possibilidade daquela carta ser usada novamente, por isso os jogadores *podem* retomar as cartas usadas e embaralhar seu naipe depois de jogarem 4 cartas e **devem** fazer isso depois de jogar a 8ª carta. O mestre sempre usará cartas até ter gasto 8, neste ponto ele deve embaralhar.
